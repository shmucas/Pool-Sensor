Wiring & schematic

All four sensors share two buses off the 40-pin header — a single GPIO for 1-Wire, and I²C (2 pins) for the ADS1115, which then fans out to three analog channels.

Flow cell note The pH and ORP probes both need to sit in moving water, not a dead-end pipe — tap a small PVC tee off the return line near the IntelliChlor so both electrode tips stay wetted. Keep the BNC connectors inside the enclosure; only probe body + cable exit through a gland.

05

Roadmap

One sensor at a time, each checked against an independent manual reading before moving on.

Stage 0 Foundations
Order hardware. Flash Raspberry Pi OS, set up SSH + WiFi, enable I²C and 1-Wire. Confirm the Pi stays connected from inside the pump enclosure.
Stage 1 Temperature only verify vs. thermometer
Wire just the DS18B20 and read it locally — validates mounting and the read loop with the simplest sensor first.
Stage 2 Add pressure verify vs. filter gauge
Wire the ADS1115 + transducer into the filter's gauge port. Compare Pi-reported PSI against the analog gauge already on the filter.
Stage 3 Add pH verify vs. test strips
Build the flow-cell tap, wire the pH probe, calibrate with buffer solutions, cross-check over a few days.
Stage 4 Add ORP verify vs. chlorine test
Same pattern as pH — wire, calibrate against a reference solution, cross-check against a manual test.
Stage 5 Ship to the dashboard verify end-to-end
Next.js API route + Supabase table; Pi POSTs all four readings on an interval; dashboard shows live + historical values.
Deferred Equipment sensing, alerts, automation
SCT-013 current clamps for pump/motor on-off state, alerting, and control — intentionally out of scope until the sensor data above is proven accurate.
Pool Sensor — internal build reference MVP scope: monitor only, no automation
